### Install
Clone this project

```
> git clone https://github.com/Ichizza/Chizza-md
> cd Chizza-md
```

Install the script:

```
> npm install
```

### Usage
1. Run The Script

```
> npm start
```
Or
```
> node index.js
```


2. Stop The Script

